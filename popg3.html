This model allows two populations to evolve by drift at five different loci. The two populations have identical starting frequencies at all loci (0.5).

Fst is a measure of differentiation between populations. If allele frequencies are identical, Fst = 0.

If populations have completely different alleles (eg A in one population, a in the other), then Fst will be equal to 1.0

The initial Fst is 0 because both populations start at the same frequency (0.5) for both alleles at each locus.

If both populations fix the same allele, Fst cannot be calculated as it would involve division by 0. 'NaN' will be displayed.
